
# Python Basics

## What is Python?

Python is a high-level, interpreted, and general-purpose programming language. It is easy to learn because of its simple and readable syntax.

## Why Learn Python?

- Easy to read and write
- Open source
- Cross-platform
- Large community support
- Widely used in Data Science, AI, Web Development, and Automation

## Applications of Python

- Data Science
- Machine Learning
- Artificial Intelligence
- ETL
- Automation
- Web Development

## First Python Program

```python
print("Hello, World!")
```

### Output

```
Hello, World!
```

## Key Points

- Python is an interpreted language.
- Python is dynamically typed.
- Indentation is mandatory.
- Python has a rich ecosystem of libraries.
